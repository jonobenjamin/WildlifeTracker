# App Assets

## Required Icons

The app needs the following icon files in this directory:

### Required Files:
- `icon.png` - Main app icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (32x32px)
- `splash-icon.png` - Splash screen icon (512x512px)

### Creating Icons:

1. **Simple Option**: Use Expo's default icons (app will still run)
2. **Custom Option**: Create wildlife-themed icons:
   - Use green/blue colors for nature theme
   - Include wildlife symbols (binoculars, paw prints, trees)
   - Keep it simple and recognizable at small sizes
